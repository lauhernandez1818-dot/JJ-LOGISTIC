# JJ LOGISTIC & JJ ASOCIADOS — Landing

Landing de conversión para JJ LOGISTIC y JJ ASOCIADOS. Next.js 14 (App Router), Tailwind CSS y Framer Motion. Branding negro y dorado metálico.

## Cómo ejecutar

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build producción

```bash
npm run build
npm start
```

## Estructura

- `app/` — Layout, página principal y estilos globales
- `components/` — Navbar, Hero, Dual-Hub, ServiceGrid, GlobalMap, ContactForm, GoldButton, Footer
- `lib/constants.ts` — Direcciones, servicios y textos

## SEO

Metadata orientada a: Logística USA-Venezuela, Agenciamiento Aduanal, JJ Logistic, Doral, Puerto Cabello.
