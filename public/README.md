# Imágenes del sitio

## Logo (branding)

Coloca aquí la imagen de tu logo/marca:

- **Archivo:** `logo.png` (o `logo.webp`)
- **Uso:** Aparece en la barra superior (navbar) y sustituye el texto del nombre.
- Si no existe el archivo, se muestra el nombre "JJ LOGISTIC PLUS | JJ ASOCIADOS" en texto.

Recomendación: logo horizontal, fondo transparente o oscuro, altura aprox. 120–150 px.

## Fotos de servicios

Crea la carpeta `services` y dentro coloca las fotos que quieras para cada servicio (aviones, camiones, barcos con carga, helicópteros, puertos, etc.). Los nombres deben ser exactamente:

- `agenciamiento.jpg` — Agenciamiento Aduanal
- `importacion.jpg` — Logísticas de Importación
- `maquinaria.jpg` — Maquinaria Pesada y Grúas
- `transporte.jpg` — Transporte Terrestre y Marítimo (camiones, barcos)
- `petroleros.jpg` — Productos Petroleros
- `navieras.jpg` — Navieras (barcos, aviones)
- `escolta-inspeccion.jpg` — Escolta e Inspección Técnica

Si falta una foto, se muestra un icono por defecto. Puedes usar .jpg o .webp.
